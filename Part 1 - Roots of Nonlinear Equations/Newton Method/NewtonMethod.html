Newton's Method: A Powerful Tool for Finding Roots

The Newton Raphson Method, sometimes referred to as Newton's Method. A useful method for numerically solving equations is the Newton Method. The roots of real-valued functions are most frequently approximated using it. The term Newton Rapson Method comes from the development of the method by Joseph Raphson and Isaac Newton. Where typical algebraic approaches would not work, this iterative approach is especially helpful for solving nonlinear problems.

History

Sir Issac Newton's unpublished book "Method of Fluxions" provided the first information on his late 17th-century development of Newton's Method. Separately, Joseph Raphson wrote "Analysis aequationum universalis," a work that improved and published a comparable method in 1690. A key idea in the solution of nonlinear equations in physics and engineering is the use of iterative approximation, which is what Newton's Method does. It has undergone substantial evolution over the ages, becoming more effective and applicable to complex systems as a result of developments in computer technology and numerical analysis. This has reinforced its position as a foundational numerical approach.

What is Newton's Method?

Finding the roots of a function f(x), or the values of x for which f(x) = 0, is done iteratively using Newton's Method. The procedure produces a series of progressively precise root estimations, starting with an initial guess, x0.

The Algorithm

The Newton-Raphson iteration formula, given a function f(x) and its derivative f'(x), is: 

xn+1 = xn - f(xn) / f'(xn)

Here’s a step-by-step breakdown of the algorithm:

  1. Initial Guess: Choose an initial guess x0.
  2. Iteration: Apply the iteration formula to find the next approximation.
  3. Convergence Check: Repeat the process until the difference between successive approximations is smaller than a predetermined tolerance level, indicating convergence.

Example

Consider the function f(x) = x2 - 2. We aim to find the root of this function, which is √2.

  1. Function and Derivative: f(x) = x2 - 2, f'(x) = 2x.
  2. Initial Guess: Let's start with x0 = 1.

Applying the Newton-Raphson formula:

xn+1 = xn - (xn2 - 2) / (2xn)

For the first iteration (n=0):

x1 = 1 - (12 - 2) / (2 * 1) = 1 - (-1/2) = 1.5

For the second iteration (n=1):

x2 = 1.5 - (1.52 - 2) / (2 * 1.5) = 1.5 - 0.25 / 3 = 1.5 - 0.0833 ≈ 1.4167

Continuing this process, we quickly converge to the root √2 ≈ 1.4142.

Why Use Newton's Method?

Advantages

  • The convergence of the procedure to the root is among the quickest.
  • Its straightforward formula makes it simple to program. It is applied to enhance a root discovered by alternative means.
  • To determine the root, just one guess is needed.
  • Since the method's derivation is more straightforward, it is simpler to comprehend its behavior, including when it should converge and diverge.

Disadvantages

  • The choice of x0 significantly affects convergence. A poor initial guess can lead to slow convergence or divergence.
  • The method requires the derivative f'(x), which might not be easy to obtain for all functions.
  • If f(x) has multiple roots, the method converges to the one closest to the initial guess, which may not always be the desired root.

Practical Implementation

Newton's Method Implementation in C++:

            
#include<bits/stdc++.h>
#define EPSILON 0.001
using namespace std;

double func(double x)
{
	return x*x*x - x*x + 2;
}

double derivFunc(double x)
{
	return 3*x*x - 2*x;
}

void newtonRaphson(double x)
{
	double h = func(x) / derivFunc(x);
	while (abs(h) >= EPSILON)
	{
		h = func(x)/derivFunc(x);
		x = x - h;
	}

	cout << "The value of the root is : " << x;
}

int main()
{
	double x = -20;
	newtonRaphson(x);
	return 0;
}

Conclusion

Newton's Method, which combines simplicity and speed, is a potent method for locating equation roots. For many numerical problems, its fast convergence gives it the best option; nevertheless, the function's nature and the first guess must be carefully considered. Whether you're doing physics, engineering, or arithmetic issues, knowing Newton's Method may greatly improve your problem-solving abilities. If you're keen to learn more, try varying the functions, first estimates, and tolerance levels to observe directly how they affect the method's accuracy and convergence.
